# AMA site — структура в стиле FrontDocs

Проект перестроен по образцу ветки `feature/dima-branch` репозитория FrontDocs:

- `base/` — базовые стили (`reset.css`, `variables.css`)
- `theme/` — тема проекта (`theme.css`)
- `js/` — клиентские скрипты, разделённые по подпапкам `core`, `data`, `utils`, `pages`
- `pages/` — отдельные HTML-страницы серий
- `supportPage/` — отдельная страница поддержки
- `images/` — изображения
- `docs/` — документы
- `main.css` — единая точка подключения стилей через `@import`

## Что изменено

- Убраны старые папки `styles/`, `scripts/`, `assets/`
- Все HTML переведены на подключение через `main.css` и отдельные JS-файлы из папки `js/`
- Починено мобильное меню на главной странице
- Добавлена отдельная страница `supportPage/support.html`
- Обновлены все относительные пути к картинкам и PDF
- Логика главной и страниц серий собрана в одном читаемом JS-файле
